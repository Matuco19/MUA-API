# Security Policy

## Supported Versions

We only support the following versions of MUA-API:

| Version | Supported          |
| ------- | ------------------ |
| 0.9   | :white_check_mark: |
| < 0.1   | :x:                |

Only supported versions will receive vulnerability updates and user support.

## Reporting a Vulnerability

To report a vulnerability, open an Issue in the repository showing about the vulnerability and ways to reproduce on our environment, this will make it easier for us to find your report!
